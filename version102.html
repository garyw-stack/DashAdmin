Version 1.0.2 Notes

Discount Functionality Update

Updates to the Point of Sale Discount Functionality.

  • Minor bug fix to round discount to the closest second decimal place
  • Update Discounts to only apply to Point items and prevent use on Non-Point items
  • Update Receipt layout to only display Discount Value fields when a discount is present

Report Generation Updates

Updated the Layout and added additional fields for reports (Daily, Monthly, and Custom Ranges) to mirror current Report layouts
